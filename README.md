# Cipherbound

## Educational Escape Room Game

Cipherbound is a two-player cooperative educational escape room game centered around programming, logic, debugging, and core computing concepts. The game is designed to reinforce knowledge in Information Technology, Computer Science, and Information Systems through interactive puzzle-solving and teamwork.

Players progress through a series of rooms by solving coding-related challenges under pressure. Each completed challenge unlocks new areas, reveals clues, and advances the players toward escaping the simulation.

---

## Game Concept

Cipherbound combines the mechanics of an escape room with educational programming challenges. The game encourages players to communicate, collaborate, and apply technical problem-solving skills in order to progress.

The experience focuses on:

* Programming logic
* Debugging
* Problem-solving
* Decoding challenges
* Team communication
* Cooperative gameplay

Each room introduces a different challenge that tests the players’ understanding of computing concepts while promoting teamwork and critical thinking.

---

## Gameplay Overview

The game begins with two players separated into different rooms. Each player must independently solve puzzles and gather information before eventually reuniting to complete cooperative challenges together.

As players advance:

* Doors unlock only after solving challenges
* Rooms contain programming or logic-based tasks
* Successfully completed rooms provide keys or hints
* Clues from previous rooms help solve future challenges
* Communication between players becomes essential

The final stage requires both players to collaboratively construct an object-oriented programming solution in order to escape.
